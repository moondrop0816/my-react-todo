# My React Todo
리액트 기반 투두 리스트 만들기


## ⚙️ 사용한 기술 / 라이브러리
- React
- CRA
- Redux
- Redux-Thunk
- Axios
- shortid
- Styled-Components
- json-server
- Google material icons
